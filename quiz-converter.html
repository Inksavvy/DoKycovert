Document Format Guide

1
Number questions: 1. text or 1) text
2
Options on separate lines: A. text / A) text / (A) text — labels stripped automatically.
3
Options can share one line: (A) cat (B) dog (C) fish (D) bird — split automatically.
4
Correct answer — underline: Underline the correct option in Word. Detected automatically.
5
Correct answer — answer key: Include 1. B 2. C 3. A anywhere, or under a heading like Answers: / Answer Key
6
If no correct answer found, the correct column is left blank.
